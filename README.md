## Getting Started

- 설치
>
> ```bash
> npm install
> # or
> yarn install
> ```

- 개발실행
>
> ```bash
> npm run dev
> # or
> yarn dev
> ```

- 운영실행
>
> ```bash
> npm run build && npm run start
> # or
> yarn build && yarn start
> ```
